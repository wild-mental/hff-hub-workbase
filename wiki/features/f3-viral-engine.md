## 한 줄 정의

**F1·F2 결과를 인스타·카톡·트위터에 그대로 붙일 수 있는 "비주얼 공유 카드(9:16, 1:1, 4:5)"를 1-Tap으로 생성하는 바이럴 엔진.**

## 해결 Pain

| Pain ID | 페르소나 | 동작 |
|---|---|---|
| **CJM-3** | A2, C1 | 비교 결과의 공유·설득 부담 → 1-Tap 격조 카드 |
| **ADJ-3** | A2 | 공유 콘텐츠의 비주얼·격조 부담 |
| **EXT-4** | E1, C2 | 가족·지인 공유 (단톡방용 포맷) |

## 핵심 AC

| ID | 기준 |
|---|---|
| F3-AC-1 | 1-Tap 생성: F1/F2 결과 페이지에서 1번 탭 → 카드 미리보기 |
| F3-AC-2 | 비율 3종 동시 생성: 9:16 (스토리), 1:1 (피드), 4:5 (포스트) |
| F3-AC-3 | 카드 안에 출처(식약처 #####) + 워터마크 (앱 도메인) |
| F3-AC-4 | 카드에서 앱 진입 시 단축 URL → 동일 비교 화면으로 진입 |
| F3-AC-5 | 카톡 단톡방용 텍스트+썸네일 포맷 별도 제공 |
| F3-AC-6 | 카드 생성 시간 ≤ 2초 (P95) |
| F3-AC-7 | UTM 자동 부착으로 공유 → 진입 추적 |

## NFR / 임계치

| 항목 | 임계치 |
|---|---|
| 카드 생성 응답 | ≤ 2초 (P95) |
| 1-Tap 공유율 (KPI) | ≥ 5% (전체 비교 결과 대비) |
| 공유 → 진입 전환율 | ≥ 12% |
| 디자인 토큰 일관성 | 100% (브랜드 가이드 준수) |
| 이미지 품질 | 1080×1920 PNG, JPEG 70% 품질 옵션 |

## 결론·시사점

- F3는 **CAC를 떨어뜨리는 유일한 자연 진입 채널**. F3 없이는 [`../concepts/q4-q1-flywheel.md`](../concepts/q4-q1-flywheel.md)가 작동 안 함.
- F3 카드의 디자인 = 본 브랜드의 시각 정체성. → MVP 0~6주에 디자인 토큰 픽스 후 변경 신중.
- "1-Tap 공유율 ≥ 5%" 가 본 사업 바이럴 가능성의 1차 KPI.
- A2 1명의 공유 카드가 평균 1.7명을 데려옴 → F3 ROI = 다른 어떤 광고 채널보다 높음.
