## 한 줄 요약

Porter의 가치사슬을 본 도메인(데이터 중개·정보 큐레이션)에 맞춰 재정의. **가치는 물리적 제품 흐름이 아니라 데이터 흐름(원본 → 정제 → 가공 → 신뢰 → 의사결정 지원)에서 발생**한다는 명제.

## 핵심 포인트

- **주활동 5종 재해석:** Inbound→데이터 정제, Operations→로직 엔진, Outbound→공유 카드, M&S→신뢰 마케팅, Service→데이터 신뢰 시스템.
- **지원활동 4종:** 1인+LLM 운영, Vibe Coding, LLM/RAG 기술, 제휴 API 조달.
- **자체 재고 0%, 변동비/매출 < 25% (추정).**
- **수익 단계:** 단기 제휴 수수료 → 중기 데이터 라이선싱 → 장기 신뢰 친화적 광고.
