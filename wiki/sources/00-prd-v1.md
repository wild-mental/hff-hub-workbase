## 한 줄 요약

HFF-Hub MVP의 비전·타깃·가치 제안·기능 정의·KPI·리스크·로드맵을 정리한 종합 PRD. 본 위키 전체의 최상위 의사결정 문서.

## 핵심 포인트

- **비전:** "비교 노가다와 죄책감으로부터의 해방."
- **MVP 4기능:** F1 HFF-Hub Engine · F2 Anti-BS Dashboard · F3 Viral Engine · F4 Data Trust System.
- **North Star:** TTC (Time to Calculation) ≤ 5초.
- **타깃 페르소나 (Phase 1~2):** C1 한정훈 · C2 박소연 · A2 정수빈.
- **로드맵 4 Phase:** 6개월·12개월·24개월·36개월 단위.
- **리스크 R1~R5:** 채널 API 변경, LLM 환각, 식약처 규제, 신고 폭주, 빅테크 진입.
