## 한 줄 요약

약사 자격을 가진 유튜버/인플루언서가 직접 영양제를 비교·추천하는 콘텐츠 형태. **개인의 권위로 신뢰를 만들지만 데이터 검증·갱신·중립성에 한계.**

## 규모·BM

| 항목 | 값 |
|---|---|
| BM | 광고/협찬 + 자체 PB/공구 |
| 강점 | "사람"의 신뢰, 스토리텔링, 트렌드 즉응성 |
| 약점 | 1인 의견의 편향, 갱신 비주기적, 특정 브랜드 협찬 의심 |

## 우리와의 차별점

| 동작 | 약사 유튜버 | 우리 |
|---|---|---|
| 신뢰 원천 | 개인의 자격·권위 | 데이터 + 인간 검수 하이브리드 |
| 갱신성 | 영상 1회 (정적) | 24h 단위 자동 (동적) |
| 검증 가능성 | 개인 의견 (반박 어려움) | 출처/산식/신고 (반박 가능) |
| 광고 분리 | 협찬 표기 의무 (Q&A 어려움) | 광고와 비교 결과 알고리즘 분리 |

## 위협/기회

### 위협

- 가장 강력한 대체재. "이 약사가 추천했으면 됐다" 신뢰 종결
- 우리 데이터를 콘텐츠 소재로 가져가면서 출처 미표기 가능

### 기회

- 약사 유튜버의 신뢰 + 우리의 데이터 정제 결합 → **우리 약사 자문단으로 영입** (KSF-4)
- "1인 의견" vs "데이터 + 다인 검수" 구도 → 장기 신뢰 우위
- 약사 유튜버의 콘텐츠가 SEO를 키우면 우리 도메인 권위가 함께 상승 (영양제 카테고리 검색 트래픽 증가)
