## 한 줄 요약

건강 목표 기반 영양제 추천 앱. 사용자 설문 → AI 추천 + 제휴 판매. **추천(Recommender) 동작에 머물러 있어 본 사업의 정제(Refiner) 동작과 직접 충돌하지 않는다.**

## 규모·BM

| 항목 | 값 (추정) |
|---|---|
| MAU | 약 30~50만 (⚠️) |
| BM | 추천 → 제휴 수수료 + 자체 PB 일부 |
| 시리즈 | Series B (확인 필요) |
| 강점 | 설문 UX, 사용자 데이터 누적 |
| 약점 | 채널 단가 정규화 ✗, 광고 검증 ✗ |

## 우리와의 차별점

| 동작 | 필라이즈 | 우리 |
|---|---|---|
| 핵심 동작 | "이걸 사라" 추천 | "이건 이런 의미" 정제 |
| 가격 비교 | 단일 채널 위주 | 다채널 1일 단가 정규화 |
| 광고 검증 | 없음 | F2 Anti-BS 핵심 |
| 출처 노출 | 약함 | F4 Data Trust 의무 |
| 수익 정렬 | 추천 ↔ 판매 (이해 충돌) | 정제 = 사용자 편 (충돌 약함) |

## 위협/기회

### 위협

- 기존 사용자 베이스에 단가 비교 기능을 후행 추가하면 우리 차별점 약화
- 자체 PB 확장 시 데이터 신뢰성 문제 → 시장 전체 인식 악화 (간접 피해)

### 기회

- "추천 앱은 광고 같다"는 사용자 불신을 우리 신뢰 자산으로 흡수
- 필라이즈에서 이탈한 C1·E2 페르소나가 첫 진입 사용자
- 같은 "건기식 앱" 카테고리지만 동작이 달라 직접 출혈 경쟁 회피
