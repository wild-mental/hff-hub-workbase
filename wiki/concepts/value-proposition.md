## 정의

본 프로젝트의 페르소나별·기능별 가치 제안을 1장으로 압축한 시트. 모든 마케팅·랜딩·앱 카피의 상위 가이드.

## 한 줄 가치 제안

> **"수동 엑셀 비교와 뒷광고 필터링에 지친 건기식 소비자에게,
> 1초 만의 1일 단가 환산과 의학 팩트체크를 제공하는 단 하나의 신뢰 가능한 대리인."**

## 페르소나별 Pain → Gain → Proof

| 페르소나 | 핵심 Pain | Gain (얻는 것) | Proof (믿게 하는 근거) |
|---|---|---|---|
| **C1** | "엑셀 비교 30분이 너무 비효율적" | 1초 안의 1일 단가 비교 | F1 실시간 환산 + 식약처 권장량 노출 |
| **C2** | "이 광고 진짜야? 내 가족에게 줘도 돼?" | 광고 → 검증 → 안심 | F2 광고 카피 즉시 검증 + 식약처 출처 |
| **A2** | "공유할 때 자신감이 없어" | 트렌디한 1-Tap 공유 카드 | F3 비주얼 공유 카드 + 트렌드 점수 |
| **E1** | "복잡해서 못 쓴다" | 한 화면에 결론 | F1 단순 모드 + 큰 글씨 + 음성(Phase 3) |
| **E2** | "데이터 출처를 모르겠다" | 모든 숫자에 출처/산식 | F4 데이터 신뢰 시스템 |
| **N1** | "지금 먹는 게 아깝다" | 같은 성분의 합리적 대안 | F1 가격비교 + F2 광고 검증 = 합리화 트리거 |

## 본 프로젝트에서 의미

### 메시지 계층 구조

1. **최상위 (Why):** "비교 노가다와 죄책감으로부터의 해방."
2. **중간 (What):** "1일 단가와 광고 진위를 1초 안에."
3. **하위 (How):** "F1·F2·F3·F4 4개 기능."

### 페르소나별 카피 톤

| 페르소나 | 톤 | 예시 카피 |
|---|---|---|
| C1 | 효율·숫자 | "오메가3 1일 1,200원 → 채널 평균 대비 32% 절약" |
| C2 | 안심·검증 | "이 광고 카피, 식약처 표시 기준 위반 가능성 ⚠️" |
| A2 | 트렌디·공유 | "MZ가 진짜 먹는 비타민 5종 — 1-Tap 공유" |
| E1 | 단순·신뢰 | "이 영양제, 안전하고 적정 가격입니다" |
| E2 | 출처·산식 | "단가 산출: 식약처 #2024-152, 1일 권장 1,000mg" |
| N1 | 합리화·전환 | "지금 드시는 ○○○과 같은 성분, 35% 저렴" |

## 핵심 차별화 (vs 경쟁사)

| 경쟁 카테고리 | 그들의 가치 제안 | 우리의 차별 |
|---|---|---|
| 비교 앱 (필라이즈) | "건강 목표별 영양제 추천" | 우리는 추천이 아니라 **검증**. 광고 차단 + 채널 단가 정규화 |
| 가격 비교 (에누리) | "최저가 검색" | 우리는 최저가가 아니라 **1일 복용량 단가**. 단위 표준화 |
| 약사 유튜버 | "전문가 추천 조합" | 우리는 1인의 의견이 아니라 **데이터 + 전문가 검수 하이브리드** |
| 홈쇼핑/D2C | "스토리텔링 + 한정 할인" | 우리는 **광고 카피 자체를 검증하는 메타 레이어** |

## 결론·시사점

- 우리의 진짜 가치 제안은 **"또 하나의 추천 앱"이 아니라 "메타 레이어(검증·재정리)"**라는 것이 모든 카피의 출발점.
- F1이 도구라면 F2·F4는 **자세(Posture)** 다. "팔지 않는다, 검증한다" — 이것이 신뢰 자산을 만든다.
- 자세한 페르소나×기능 매트릭스는 raw/06 참조.
