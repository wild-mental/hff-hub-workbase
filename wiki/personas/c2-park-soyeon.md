## 한 줄 요약

**책임감 동력 엔진** — 본인 또는 가족의 건강 신호로 처음 영양제를 진지하게 사보려는 30대 여성. **MVP 진입 트래픽의 핵심.**

## 프로필

| 항목 | 값 |
|---|---|
| 나이 | 30대 |
| 직업 | 직장인 또는 워킹맘 |
| 가구 | 2~4인 가구 (아이/부모 동반 가능) |
| 디지털 친숙도 | 높음 (카톡/유튜브/인스타) |
| 세그먼트 | Q4-A (인지 低 / 경험 無, 안전 동기) |
| 1인 LTV (추정) | 약 5.7만 원 (장기 전환 시 ↑) |

## 핵심 Pain

| ID | Pain |
|---|---|
| **CORE-2** | 광고/뒷광고 신뢰 붕괴 — "이거 진짜야?" |
| **CORE-3** | 성분 해석 불가 — 마케팅 용어를 못 알아들음 |
| **CORE-4** | 결정 기준 부재 — "이 가격이 적정한가?" |
| **CJM-1** | 인지 단계의 막연한 불안 — "내가 골랐다가 잘못되면" |
| **EXT-3** | 부작용·약물 상호작용 우려 (특히 가족용) |

## Goal

> "광고 노이즈 없이 검증된 정보로, 가족과 나에게 줘도 되는 첫 영양제를 안전하게 고르고 싶다."

## 감정 곡선

```
  ① 인지 ─★── ② 탐색 ─★─── ③ 결정 ──★ ④ 복용 ── ⑤ 재구매
  불안↑     좌절·죄책감    회의·확인     기대    
  ★ 트리거 시점        ★ 깊은 골짜기   ★ 마지막 한 번 더 의심
```

## JTBD Job Story

> When 건강 검진에서 안 좋은 신호를 받거나 가족이 아플 때,
> I want 광고 노이즈 없이 검증된 정보로 첫 영양제를 안전하게 고르고,
> so I can "이거 진짜 효과 있나?"라는 죄책감 없이 가족과 나를 챙길 수 있다.

## AOS / DOS

- 점유 사분면: **Q1 (혁신 기회)** + 일부 **Q2 (차별화)**
- 핵심 AOS: CORE-2 (9.0), CORE-4 (8.5), CJM-1 (7.9)

## 카피 톤 가이드

- 안심·확신 키워드: "검증된", "식약처 기준", "안전하게"
- 의학적 단정 회피: "치료/예방" 단어 절대 금지 (건기식법)
- 감정 단어 사용 가능 ("죄책감 없이", "안심하고")

## 결론·시사점

- C2는 **Phase 2 (6~12개월) 메인 타깃**. SEO 키워드는 "[증상] 영양제 추천", "[성분명] 광고 진짜?".
- C2 → C1 전환율이 본 사업 장기 LTV의 핵심. F1·F2·F4 패키지가 안 깨져야 전환됨.
- C2의 첫 인상은 F2 Anti-BS Dashboard의 광고 검증 결과 1건이 90% 결정 → MVP에서 시각화 품질 1순위.
