## 한 줄 요약

**가족 보호 동력 엔진** — 자녀 또는 약사가 추천한 영양제를 본인이 직접 사고 싶어하는 50대 후반~60대 여성. **Phase 3 확장 타깃.**

## 프로필

| 항목 | 값 |
|---|---|
| 나이 | 50대 후반 ~ 60대 |
| 직업 | 주부, 자영업자 |
| 가구 | 2~4인 가구 (배우자, 성인 자녀) |
| 디지털 친숙도 | 낮음 (카톡 OK, 앱 설치 어려움) |
| 세그먼트 | Q4-B (인지 低 / 경험 無, 단순성·관계 동기) |
| 1인 LTV (추정) | 약 5.7만 원 |

## 핵심 Pain

| ID | Pain |
|---|---|
| **EXT-1** | 효능 불확실성 — "이거 진짜 효과 있는 거 맞아?" |
| **CJM-1** | 인지 단계의 복잡성 — "내가 결정할 수 있을까?" |
| **CORE-3** | 성분 해석 불가 — 영문/한문 용어 두려움 |
| **EXT-4** | 가족·지인 공유/추천 어려움 — 자녀에게 다시 묻기 |

## Goal

> "복잡한 비교 없이, 자녀에게 다시 묻지 않고도 안전하고 적정한 영양제를 스스로 결정하고 싶다."

## 감정 곡선

```
  ① 인지 ── ② 탐색 ─★── ③ 결정 ── ④ 복용 ── ⑤ 재구매
  의존↑      복잡 → 포기   의존↑      관성       자녀 의존
  (자녀에게 물음)  ★ 디지털 좌절   (자녀 대행)
```

★ = 기존 앱들이 모두 실패하는 지점

## JTBD Job Story

> When 자녀가 영양제를 추천했거나 약국에서 권유받았을 때,
> I want 복잡한 비교 없이 "이거 안전하고 값 적당한지" 한 화면으로 확인하고,
> so I can 자녀에게 다시 묻지 않고 스스로 결정해 가족을 챙길 수 있다.

## AOS / DOS

- 점유 사분면: **Q1 + Q3 일부** — 시급성 中, 미해결도 高
- 핵심 AOS: EXT-1 (7.8), CJM-1 (7.2)

## 관련 기능 · CJM 단계

| 우선순위 | 기능 | CJM | 동작 |
|---|---|---|---|
| ★★★ | F1 (단순 모드) | ② | 결과 한 줄: "안전·적정" |
| ★★ | F4 Data Trust | ③ | 식약처 공식 인증 표시 |
| ★ | (Phase 3) 음성 UI | ②,③ | "오메가3 알려줘" |
| ★ | (Phase 3) 약사 큐레이션 라이브 | ② | 사람 목소리의 신뢰 |

## 카피 톤 가이드

- 큰 글씨, 단순 색상(빨/녹 신호등 메타포)
- "안전합니다", "적정 가격입니다" 결론 우선
- 영문/한문/약어 회피 (오메가3 → "혈관 영양제")
- "자녀와 공유" 버튼을 가장 크게

## 결론·시사점

- E1은 **Phase 3 (12~24개월) 타깃**. MVP 시점은 단순 모드 토글로 부분 대응.
- E1은 자녀(C1/C2)가 대행 사용하는 케이스가 70%+ → C1·C2 만족이 곧 E1 만족.
- 음성 UI / 약사 라이브는 비싼 기능이지만 E1 만족도가 시장 권위를 만든다 (50대+ 진입 = "엄마도 쓰는 앱" 신뢰).
- E1이 못 쓴다면 그 앱은 절대 메인스트림이 못 된다 → UX 단순성의 최종 시험대.
